4. Compute API Extensions Examples

GET /v1.1/extensions

Shows the installed extensions to the API, which enable more requests and actions for the Compute server.

Curl example: 

curl -D - -H "X-Auth-Token: 8528179e-0d07-44e9-b972-7427336c94cb" -X 'GET' -v http://demo.rcb.me:8774/v1.1/9/extensions
