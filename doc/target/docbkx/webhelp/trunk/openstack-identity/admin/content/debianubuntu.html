Debian/Ubuntu

  1. Add the Keystone PPA to your sources.lst:

    ::

    $> sudo add-apt-repository ppa:keystone-core/trunk $> sudo apt-get update

  2. Install Keystone:

    ::

    $> sudo apt-get install keystone
